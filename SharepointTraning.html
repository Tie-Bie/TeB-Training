🏢 หลักสูตรและเนื้อหา

🔹 Module 1: แนะนำ SharePoint Online

วัตถุประสงค์: เข้าใจว่า SharePoint คืออะไร ใช้ทำอะไรได้บ้าง

📚 เนื้อหา:

  • ความสามารถหลักของ SharePoint
  • ความแตกต่างระหว่าง SharePoint และ OneDrive / Teams
  • ตัวอย่างการใช้งาน SharePoint ในองค์กร

🎯 กิจกรรมฝึกปฏิบัติ:

  • เข้าเว็บไซต์ SharePoint
  • สำรวจหน้าต่าง UI ของ SharePoint

🔹 Module 2: การสร้างและจัดการ Team Site / Communication Site

วัตถุประสงค์: สร้างพื้นที่การทำงานร่วมกันในองค์กร

📚 เนื้อหา:

  • ความแตกต่างระหว่าง Team Site และ Communication Site
  • การกำหนดสิทธิ์ในการเข้าถึง
  • การเชื่อมโยงกับ Microsoft Teams

🎯 กิจกรรมฝึกปฏิบัติ:

  • สร้าง Team Site ของตนเอง
  • เชิญสมาชิกเข้า Site
  • ตั้งค่าการอนุญาต (Permissions)

🔹 Module 3: การจัดการเอกสารและ Library

วัตถุประสงค์: ใช้งาน Document Library อย่างมีประสิทธิภาพ

📚 เนื้อหา:

  • การสร้าง Document Library
  • การอัปโหลด / แชร์ / เวอร์ชันเอกสาร
  • การกำหนดสิทธิ์รายไฟล์หรือโฟลเดอร์
  • Metadata และการจัดหมวดหมู่เอกสาร

🎯 กิจกรรมฝึกปฏิบัติ:

  • สร้าง Document Library
  • อัปโหลดเอกสารหลายไฟล์
  • ตั้งเวอร์ชันเอกสารและลอง Check in / Check out

🔹 Module 4: การสร้างหน้า Page และ Web Parts

วัตถุประสงค์: สร้างหน้าข้อมูล (Pages) ที่ดึงดูดและใช้งานง่าย

📚 เนื้อหา:

  • การสร้างหน้า News / Site Page
  • การใช้ Web Parts เช่น Text, Quick Links, Image, List, Embed
  • การออกแบบหน้า Dashboard

🎯 กิจกรรมฝึกปฏิบัติ:

  • สร้างหน้าเว็บประชาสัมพันธ์ข่าวสาร
  • ใส่ Web Part ที่จำเป็น
  • แชร์หน้าเว็บให้เพื่อนร่วมทีมดู

🔹 Module 5: การสร้าง List และ Automation ด้วย Power Automate

วัตถุประสงค์: จัดการข้อมูลและเอกสารอย่างอัตโนมัติ

📚 เนื้อหา:

  • การสร้าง Custom List เช่น แบบฟอร์มขอลา / แบบประเมิน
  • การสร้าง Flow พื้นฐาน (Power Automate)
  • ตัวอย่าง Automation เช่น การแจ้งเตือนเมื่อมีการอัปโหลดเอกสาร

🎯 กิจกรรมฝึกปฏิบัติ:

  • สร้าง List สำหรับเก็บคำขอลา
  • สร้าง Flow แจ้งเตือนเมื่อมีรายการใหม่

🔹 Module 6: การดูแลระบบและ Best Practices

วัตถุประสงค์: ดูแลระบบและการใช้งานอย่างถูกต้อง

📚 เนื้อหา:

  • การสำรองข้อมูล
  • การตรวจสอบกิจกรรมใน Site
  • การวางโครงสร้าง Site และ Naming Convention
  • แนวทางการใช้งาน SharePoint อย่างปลอดภัย

🎯 กิจกรรมฝึกปฏิบัติ:

  • สำรวจ Audit log
  • ออกแบบโครงสร้าง Site Map เบื้องต้น
