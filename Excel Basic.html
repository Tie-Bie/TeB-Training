🏠 Excel Intermediate : MSE-LB02 Excel Advanced : MSE-LB02

อบรมการใช้งาน Microsoft Excel Basic

สำหรับผู้เริ่มต้นจนถึงระดับกลาง - เพิ่มทักษะและความเชี่ยวชาญของคุณ โดยรวบรวมความสามารถต่าง ๆ ทั้งเรื่อง สูตรการคำนวณ การสรุปผลข้อมูลด้วยสูตรคำนวณ การทำงานกับข้อมูล การนำเสนอด้วยกราฟ เพื่อใช้ Excel แก้ไขปัญหาทางธุรกิจได้อย่างมีประสิทธิภาพ!

วัตถุประสงค์

  • ผู้อบรมสามารถบอกถึงประโยชน์ของโปรแกรม Microsoft Excel
  • ผู้อบรมสามารถสร้าง/แก้ไข ตารางเอกสารได้
  • ผู้อบรมสามารถทำการคำนวณด้วย Microsoft Excel
  • ผู้อบรมสามารถใช้สูตรเพื่อคำนวณสรุปผลของข้อมูลได้หลากหลาย
  • ผู้อบรมสามารถทำงานกับข้อมูล (Data) ทั้งเรื่องการ Sort, Filter ได้
  • ผู้อบรมสามารถสร้างกราฟแท่ง กราฟวงกลม กราฟเส้น ได้

หลักสูตรนี้เหมาะสำหรับ

  • ผู้ใช้งานทั่วไป
  • ผู้ที่ต้องการเริ่มต้นกับ Excel อย่างเป็นระบบ
  • ผู้ที่ต้องการใช้ Excel คำนวณแบบง่าย ๆ
  • ผู้ที่ต้องการใช้ Excel สร้างกราฟจากข้อมูลที่ไม่ซับซ้อน

หัวข้อการฝึกอบรม

เริ่มต้นกับ Microsoft Excel
ส่วนประกอบของโปรแกรม Microsoft Excel, ความสามารถของ Microsoft Excel, การจัดมุมมองการทำงานกับไฟล์, การจัดการไฟล์เอกสาร Excel
เทคนิคการทำงานพื้นฐาน
คีย์ลัดในการทำงาน, การเคลื่อนไปยังส่วนต่าง ๆ ของ Excel, การเลือกข้อมูลแบบรวดเร็ว, การจัดรูปแบบข้อมูลแบบรวดเร็ว
การทำงานกับวันที่และการทำงานอัตโนมัติ
การทำงานกับวันที่, เทคนิคการตรวจสอบรูปแบบวันที่, การทำงานอัตโนมัติด้วย Auto Fill และ Flash Fill, การทำ Dropdown List
การจัดรูปแบบขั้นสูง
การจัดรูปแบบเซลล์ให้เหมาะกับเนื้อหา, การจัดรูปแบบด้วยเงื่อนไข (Conditional Formatting), เทคนิคการจัดรูปแบบต่าง ๆ
การคำนวณและสูตรใน Microsoft Excel
เครื่องหมายการคำนวณ, การอ้างอิง Cell/Worksheet/Workbook, สูตร IF, การ Lock Cell, สูตรคำนวณพื้นฐาน (SUM, AVERAGE, MAX, MIN, COUNT), สูตรข้อความ, สูตรวันที่, VLOOKUP
การสร้างกราฟเพื่อนำเสนอข้อมูล
การสร้างกราฟและกำหนดโครงสร้างข้อมูล, กราฟแท่ง, กราฟวงกลม, กราฟเส้น, กราฟ 2 แกน พร้อมกรณีศึกษาต่าง ๆ
การทำงานกับข้อมูล
การเรียงข้อมูลตามลำดับ (Sorting), การกรองข้อมูล (Filter) ด้วย Auto Filter, การกำหนดเงื่อนไขต่าง ๆ, Top 10, เทคนิคการกรองข้อมูลตามต้องการ
Tips และเทคนิคขั้นสูง
เครื่องมือแยกตัวอักษร (Text to Columns), การสร้างลิสต์รายการอัตโนมัติ, การตั้งค่าเวลาป้องกันความเสี่ยง, เทคนิคการคัดลอกสูตรรวดเร็ว, การทำเลขรันอัตโนมัติ
การรักษาความปลอดภัย
การกำหนดรหัสผ่านให้ไฟล์ Excel, การป้องกัน Worksheet & Workbook (Protect Sheet, Protect Workbook)
