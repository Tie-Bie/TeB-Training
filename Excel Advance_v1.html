หัวข้อการฝึกอบรม

🚀 Microsoft Excel พื้นฐานขั้นสูง

  • ความสามารถและส่วนประกอบของ Microsoft Excel
  • ประเภทไฟล์ Excel ข้อดี และข้อจำกัด
  • คีย์ลัด Shortcut Key ที่จำเป็นต่อการทำงาน
  • การจัดรูปแบบที่สามารถทำได้อย่างรวดเร็ว

🔧 ฟังก์ชันขั้นสูง

  • สูตรการคำนวณเพื่อทำงานตามเงื่อนไข (IFS, SWITCH)
  • ฟังก์ชัน SUMIFS, COUNTIFS, AVERAGEIFS
  • ฟังก์ชัน OFFSET, INDIRECT, XLOOKUP
  • ฟังก์ชันใหม่ใน Microsoft 365 (FILTER, SORT, UNIQUE)

📈 การจัดการข้อมูล

  • การทำงานกับข้อมูลด้วย Table
  • Power Query เพื่อการเตรียมข้อมูล
  • PivotTable และ PivotChart
  • Conditional Formatting และ Data Validation

🔐 ความปลอดภัย

  • การกำหนดรหัสผ่านให้ไฟล์ Excel
  • การป้องกัน Worksheet & Workbook
  • การป้องกันไม่ให้เห็นสูตรและแก้ไข Cell

ความต้องการของระบบและพื้นฐานผู้เข้าอบรม

💻 ความต้องการของระบบ

  • Windows 11 / Windows 10
  • Microsoft Excel | Microsoft 365
  • สำหรับ Version 2019, 2021 บางคำสั่งจะไม่สามารถใช้งานได้

👨‍💼 เหมาะสำหรับ

  • ผู้ที่เคยใช้งาน Excel แต่ต้องการทำงานให้ง่าย เร็วยิ่งขึ้น
  • ผู้ที่ต้องการประยุกต์ใช้สูตรที่ซับซ้อน
  • ผู้ที่ต้องการทำงานกับข้อมูลอย่างมีประสิทธิภาพ
