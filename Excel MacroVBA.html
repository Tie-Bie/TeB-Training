Microsoft Excel Macro และ VBA

อบรมระบบงานอัตโนมัติด้วย Macro และ VBA ใน Microsoft Excel หากต้องการลดระยะเวลาในการทำงานลักษณะซ้ำ ๆ เดิม เราสามารถบันทึกกระบวนการทำงานนั้น ๆ ได้ โดยใช้ Macro ซึ่งการบันทึกดังกล่าว หากต้องการให้ Dynamic มากยิ่งขึ้น จำเป็นต้องแก้ไข Macro และปรับแต่ง code ที่เป็น Visual Basic for Application (VBA) ได้

หัวข้อการฝึกอบรม

Review Microsoft Excel

  • ส่วนประกอบต่าง ๆ ของ Microsoft Excel
  • การใช้คีย์ลัดใน Microsoft Excel อีกช่องทางในการใช้งานร่วมกับ Macro
  • การทำงานกับข้อมูลจำนวนมาก ๆ
  • การลิงก์ข้อมูลและสูตรข้าม Worksheet, Workbook
  • Advanced Featured in Microsoft Excel

การ Import/Export Data

  • Import Text File (.csv, .txt)
  • Import Excel File
  • Export Data
  • แนะนำการสรุปผลข้อมูลด้วย PivotTable
  • Dynamic PivotTable

Introduction to Macro

  • ความหมายและประโยชน์ของ มาโคร (Macro)
  • การเรียกใช้ Developer Tab และส่วนประกอบของ Developer Tab
  • การกำหนดค่าให้กับ Microsoft Office Excel เพื่อใช้งาน Macro
  • การบันทึกมาโคร (Macro)
  • การแก้ไข Macro Excel
  • รู้จักกับ Relative Reference

Visual Basic for Application (VBA)

  • โครงสร้างทางภาษา VBA
  • Visual Basic Editor (VBE)
  • Excel Object Model
  • Event, Object, Properties, Method
  • การกำหนด ตัวแปร (Variable) และขอบเขตของตัวแปร
  • วิธีการอ่านโค้ด และ เทคนิคการแก้ไขโค้ด VBA
  • การดีบั๊กโปรแกรม (Debugging)

Case Study

  • ระบบใบแจ้งหนี้ที่สามารถเก็บรายละเอียดของใบแจ้งหนี้ก่อนหน้าได้
  • การ Import Data ของการรายการขาย เพื่อสรุปข้อมูลในรูปแบบ PivotTable / PivotChart อัตโนมัติ
  • การ Export Data รายการลูกค้าอัตโนมัติ โดยกำหนดเงื่อนไขการ Export ได้
  • การใช้มาโครช่วยในการตกแต่งเอกสารอย่างรวดเร็วในพริบตา
  • เทคนิคการทำให้ระบบงานมีความยืดหยุ่น (Flexible)
