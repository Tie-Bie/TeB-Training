Google BigQuery for Business

วิเคราะห์ Big Data อย่างมืออาชีพ ด้วย SQL บน Cloud (เร็ว ประหยัด และต่อยอดได้จริง) คอร์สนี้จะทำให้คุณใช้ Google BigQuery จัดการข้อมูลขนาดใหญ่ได้จริง

  • ต้องการการลดงานซ้ำ และเพิ่มประสิทธิภาพการทำงาน
  • ต้องการการใช้ Excel เพื่อเข้าใจข้อมูลเชิงธุรกิจ
  • ต้องการต่อยอดไปสู่ Power BI หรือ Big Data ได้
  • ผู้ที่อยากย้ายสายเป็น Data Analyst / BI
📌 เริ่มวิเคราะห์ Big Data ได้จริง เพื่อทำงานเร็วขึ้นและตัดสินใจแม่นขึ้น

📚 เนื้อหาหลักสูตร (Course Outline)

Module 1: BigQuery Overview & Big Data Mindset

เข้าใจภาพรวมก่อนลงมือ — ทำให้เรียนแล้วใช้งานจริงได้

BigQuery คืออะไร และองค์กรใช้ทำอะไร
Data Warehouse vs Database ต่างกันยังไง
ทำความรู้จักและจัดการกับข้อมูล Big Data
โครงสร้าง BigQuery: Project / Dataset / Table

Module 2: เริ่มต้นใช้งาน BigQuery (Hands-on Setup)

การใช้งาน BigQuery Console
การจัดการ Project / Permissions
การสร้าง Dataset และ Table
การ Upload ข้อมูลจาก CSV / Excel / Google Sheets
Data Type ที่ควรรู้ (STRING, INT, FLOAT, DATE, TIMESTAMP)

Module 3: SQL for BigQuery (พื้นฐานที่ใช้จริงในงาน)

ทักษะหลักที่ทำให้คุณวิเคราะห์ข้อมูลได้

SELECT / WHERE / ORDER BY / LIMIT
Aggregation: COUNT, SUM, AVG
GROUP BY / HAVING / Subquery & CTE (WITH)
JOIN (หัวใจของการรวมข้อมูลธุรกิจ)

Module 4: BigQuery Functions ที่ใช้จริง (งาน Data Analyst ต้องใช้)

สูตรและเทคนิคที่ช่วยให้ Query ได้ไวและแม่นขึ้น

การตั้งคำถามกับข้อมูล (Business Questions)
การหา Insight จากยอดขาย/ต้นทุน/ลูกค้า
วิเคราะห์ KPI / Trend / เปรียบเทียบช่วงเวลา
เทคนิคการเล่าเรื่องด้วยข้อมูล (Data Storytelling)

Module 6: Dashboard & Report Design (ทำรายงานหน้าเดียวแบบใช้งานจริง)

ทักษะที่ได้: สร้าง Dashboard แบบมืออาชีพ สามารถวิเคราะห์เพื่อการตัดสินใจได้

ออกแบบ Dashboard ที่ผู้บริหารเข้าใจใน 1 หน้า
ทำ Interactive ด้วย Slicer
Best Practice Dashboard สำหรับ Excel
Template Dashboard ที่นำไปใช้ต่อได้

ทักษะที่ได้หลังเรียนจบ (Learning Outcomes)

✅ เข้าใจ BigQuery และแนวคิด Data Warehouse บน Cloud

มองเห็นภาพว่าควรใช้ Power BI เมื่อไร และช่วยองค์กรยังไง

✅ ใช้ SQL เพื่อดึงข้อมูล วิเคราะห์ และสร้างรายงาน

ออกแบบโครงสร้างข้อมูลที่รองรับการวิเคราะห์ระยะยาว

✅ จัดการข้อมูลขนาดใหญ่ได้เร็วกว่าเดิมหลายเท่า

สร้างตัววัดยอดขาย กำไร การเติบโต และตัวชี้วัดอื่นๆ

✅ ทำ Data Pipeline เบื้องต้น + สร้าง Table/View สำหรับ Dashboard

สร้างตัววัดยอดขาย กำไร การเติบโต และตัวชี้วัดอื่นๆ

✅ รวมข้อมูลหลายแหล่ง (CSVExcel/Sheets)

สร้างรายงานที่อ่านง่าย ตอบคำถามทางธุรกิจได้ในไม่กี่คลิก

✅ เชื่อม BigQuery ไปยัง Looker Studio / Power BI ได้จริง

สร้างตัววัดยอดขาย กำไร การเติบโต และตัวชี้วัดอื่นๆ
