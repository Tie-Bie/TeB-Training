Office 365 เบื้องต้นสำหรับผู้เริ่มต้น

เรียนรู้การใช้งาน Office 365 เพื่อเพิ่มประสิทธิภาพในการทำงานและการสื่อสาร

**วัตถุประสงค์:** เพื่อให้ผู้เรียนเข้าใจและสามารถใช้งานโปรแกรมหลักในชุด Office 365 ได้อย่างมีประสิทธิภาพในระดับพื้นฐาน

รายละเอียดหลักสูตร

กลุ่มเป้าหมาย:

  • ผู้ใช้งานใหม่ที่ยังไม่มีประสบการณ์กับ Office 365
  • ผู้ที่ต้องการทบทวนความรู้พื้นฐานและเรียนรู้ฟังก์ชันใหม่ๆ ของ Office 365
  • นักเรียน นักศึกษา หรือบุคคลทั่วไปที่ต้องการเพิ่มทักษะด้านดิจิทัล

โมดูลที่ 1: แนะนำ Office 365 และพื้นฐานการใช้งาน (Introduction to Office 365 & Basic Usage)

  • 1.1 ทำความรู้จัก Office 365:

    • Office 365 คืออะไร? (ความแตกต่างจาก Office แบบติดตั้ง)
    • ประโยชน์และข้อดีของการใช้ Office 365
    • ส่วนประกอบหลักของ Office 365 (Word, Excel, PowerPoint, Outlook, OneDrive, Teams)

  • 1.2 การเข้าสู่ระบบและการจัดการบัญชี:

    • วิธีการเข้าสู่ระบบ Office 365 ผ่านเว็บเบราว์เซอร์
    • การตั้งค่าโปรไฟล์และรหัสผ่านเบื้องต้น
    • การติดตั้งแอปพลิเคชัน Office 365 บนคอมพิวเตอร์และมือถือ

  • 1.3 อินเทอร์เฟซและเมนูพื้นฐาน:

    • ภาพรวมของหน้าจอหลักและเมนูต่างๆ ในแอปพลิเคชัน Office
    • แถบ Ribbon และการปรับแต่ง
    • การบันทึกและเปิดไฟล์ (Save, Save As, Open)

โมดูลที่ 2: Microsoft Word เบื้องต้น (Basic Microsoft Word)

  • 2.1 การสร้างและแก้ไขเอกสาร:

    • การเปิดเอกสารใหม่และเทมเพลต
    • การพิมพ์ข้อความและการแก้ไขเบื้องต้น
    • การตัด, คัดลอก, วาง (Cut, Copy, Paste)

  • 2.2 การจัดรูปแบบข้อความและย่อหน้า:

    • การเปลี่ยนฟอนต์, ขนาด, สี, สไตล์ (Bold, Italic, Underline)
    • การจัดตำแหน่งข้อความ (Alignments)
    • การใช้ Bullet Points และ Numbering

  • 2.3 การแทรกวัตถุพื้นฐาน:

    • การแทรกรูปภาพ
    • การแทรกตาราง
    • การแทรกรูปร่าง (Shapes)

  • 2.4 การบันทึกและส่งออกเอกสาร:

    • การบันทึกเป็นรูปแบบต่างๆ (เช่น .docx, .pdf)
    • การพิมพ์เอกสาร

โมดูลที่ 3: Microsoft Excel เบื้องต้น (Basic Microsoft Excel)

  • 3.1 ทำความรู้จักกับ Worksheet และ Cell:

    • โครงสร้างของ Worksheet (แถว, คอลัมน์, เซลล์)
    • การป้อนข้อมูลและการแก้ไขในเซลล์
    • การเลือกเซลล์, แถว, คอลัมน์

  • 3.2 การจัดรูปแบบข้อมูล:

    • การเปลี่ยนฟอนต์, สี, ขนาด
    • การจัดตำแหน่งข้อมูลในเซลล์
    • การจัดรูปแบบตัวเลข (Number Formatting)

  • 3.3 สูตรและฟังก์ชันพื้นฐาน:

    • การใช้สูตรคำนวณพื้นฐาน (SUM, AVERAGE, MAX, MIN, COUNT)
    • การใช้เครื่องหมายทางคณิตศาสตร์ (+, -, *, /)
    • การอ้างอิงเซลล์ (Cell References)

  • 3.4 การสร้างแผนภูมิ (Charts) อย่างง่าย:

    • ประเภทของแผนภูมิที่นิยมใช้ (Column, Bar, Pie, Line)
    • การสร้างแผนภูมิจากข้อมูล

  • 3.5 การเรียงลำดับและกรองข้อมูล (Sort & Filter):

    • การเรียงลำดับข้อมูล (Ascending, Descending)
    • การใช้ Filter เพื่อคัดกรองข้อมูลที่ต้องการ

โมดูลที่ 4: Microsoft PowerPoint เบื้องต้น (Basic Microsoft PowerPoint)

  • 4.1 การสร้างงานนำเสนอ:

    • การเลือก Layout และ Theme
    • การเพิ่ม, ลบ, จัดเรียงสไลด์
    • การป้อนข้อความและหัวข้อ

  • 4.2 การจัดรูปแบบสไลด์:

    • การเปลี่ยนพื้นหลังสไลด์
    • การใช้ Master Slide (เบื้องต้น)

  • 4.3 การแทรกวัตถุ:

    • การแทรกรูปภาพ, ไอคอน
    • การแทรกตาราง, แผนภูมิ (จาก Excel)

  • 4.4 การใช้ภาพเคลื่อนไหว (Animations) และการเปลี่ยนสไลด์ (Transitions):

    • การเพิ่ม Animation ให้กับวัตถุ
    • การใส่ Transition ระหว่างสไลด์

  • 4.5 การนำเสนอ (Presentation Mode):

    • การเริ่มต้นและสิ้นสุดการนำเสนอ
    • โหมด Presenter View

โมดูลที่ 5: Microsoft Outlook และ OneDrive เบื้องต้น (Basic Microsoft Outlook & OneDrive)

  • 5.1 Microsoft Outlook:

    • ภาพรวมการใช้งาน Outlook (อีเมล, ปฏิทิน, ผู้ติดต่อ)
    • การส่งและรับอีเมล
    • การแนบไฟล์
    • การจัดการปฏิทินและนัดหมายเบื้องต้น

  • 5.2 OneDrive:

    • OneDrive คืออะไร? ประโยชน์ของการจัดเก็บไฟล์บนคลาวด์
    • การอัปโหลดและดาวน์โหลดไฟล์
    • การสร้างโฟลเดอร์และการจัดระเบียบไฟล์
    • การแชร์ไฟล์และโฟลเดอร์กับผู้อื่น
    • การทำงานร่วมกันบนไฟล์เดียวกัน (Co-authoring)

โมดูลที่ 6: Microsoft Teams เบื้องต้น (Basic Microsoft Teams)

  • 6.1 ทำความรู้จักกับ Microsoft Teams:

    • Teams คืออะไร? ประโยชน์สำหรับการทำงานร่วมกัน
    • ภาพรวมของอินเทอร์เฟซ (Teams, Chat, Calendar, Files)

  • 6.2 การใช้งาน Chat และ Calls:

    • การส่งข้อความส่วนตัวและกลุ่ม
    • การโทรด้วยเสียงและวิดีโอคอล

  • 6.3 การเข้าร่วมและใช้งาน Team (ทีม):

    • การเข้าร่วม Team ที่มีอยู่
    • การโพสต์ข้อความใน Channel
    • การแชร์ไฟล์ใน Team

  • 6.4 การเข้าร่วมประชุมออนไลน์:

    • การเข้าร่วมประชุมผ่านลิงก์หรือปฏิทิน
    • ฟังก์ชันพื้นฐานในระหว่างประชุม (เปิด/ปิดไมค์, กล้อง, แชร์หน้าจอ)

โมดูลที่ 7: Microsoft Power BI เบื้องต้น (Basic Microsoft Power BI)

  • 7.1 ทำความรู้จัก Power BI:

    • Power BI คืออะไร? ประโยชน์สำหรับการวิเคราะห์ข้อมูล
    • ส่วนประกอบหลัก (Power BI Desktop, Service, Mobile)

  • 7.2 การนำเข้าข้อมูลและการสร้างรายงานอย่างง่าย:

    • การเชื่อมต่อกับแหล่งข้อมูล (Excel, CSV)
    • การสร้าง Visualization พื้นฐาน (Chart, Table)
    • การบันทึกและเผยแพร่รายงาน

โมดูลที่ 8: Microsoft SharePoint เบื้องต้น (Basic Microsoft SharePoint)

  • 8.1 ทำความรู้จัก SharePoint:

    • SharePoint คืออะไร? ประโยชน์สำหรับการจัดการเอกสารและเว็บไซต์ทีม
    • ภาพรวมของ Site, Document Library, List

  • 8.2 การใช้งาน Document Library:

    • การอัปโหลด, ดาวน์โหลด, แก้ไขไฟล์
    • การสร้างโฟลเดอร์และการจัดระเบียบ
    • การแชร์เอกสารและการควบคุมสิทธิ์

โมดูลที่ 9: Microsoft Power Automate เบื้องต้น (Basic Microsoft Power Automate)

  • 9.1 ทำความรู้จัก Power Automate:

    • Power Automate คืออะไร? ประโยชน์สำหรับการสร้าง Workflow อัตโนมัติ
    • แนวคิดของ Trigger และ Action

  • 9.2 การสร้าง Flow อย่างง่าย:

    • การสร้าง Automated Cloud Flow (เช่น เมื่อมีอีเมลใหม่, บันทึกไฟล์ลง OneDrive)
    • การสร้าง Instant Cloud Flow (เรียกใช้ด้วยตนเอง)

โมดูลที่ 10: Microsoft Power Apps เบื้องต้น (Basic Microsoft Power Apps)

  • 10.1 ทำความรู้จัก Power Apps:

    • Power Apps คืออะไร? ประโยชน์สำหรับการสร้างแอปพลิเคชันแบบ Low-code
    • ประเภทของแอป (Canvas apps, Model-driven apps)

  • 10.2 การสร้าง Canvas App อย่างง่าย:

    • การสร้างแอปจาก Template หรือจากข้อมูล (เช่น Excel)
    • การเพิ่ม Controls พื้นฐาน (Text input, Button, Gallery)
    • การบันทึกและเผยแพร่แอป
