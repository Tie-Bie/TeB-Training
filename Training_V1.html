วัตถุประสงค์

  1. ผู้อบรมสามารถอ่านค่าประไฟฟ้าอย่างถูกต้องโปรแกรม Microsoft Excel
  2. ผู้อบรมสามารถสร้างแฟ้มใหม่ การบันทึกเอกสารได้
  3. ผู้อบรมสามารถทำการคำนวณด้วย Microsoft Excel
  4. ผู้อบรมสามารถใช้สูตรพื้นฐานกับคำนวณสูตรปฏิเวียนของสูตรได้อย่างคล่องคลาย
  5. ผู้อบรมสามารถทำงานกับข้อมูล (Data) กับเรื่องการ Sort, Filter ได้
  6. ผู้อบรมสามารถสร้างกราฟแก้ไข กราฟ/วิจิตรกม กราฟสีสวก ได้

หลักสูตรนี้เหมาะสำหรับ

  • ผู้ใช้งานทั่วไป
  • ผู้ที่ต้องการเรียนพื้นที่ Excel อย่างเป็นระบบ
  • ผู้ที่ต้องการใช้ Excel ทำงานแแบบเดิมว่าง ๆ
  • ผู้ที่ต้องการใช้ Excel สร้างกราฟ/วิจิตรอบสถิติได้อย่างนาเรียน

ความต้องการของระบบ

  1. ระบบปฏิบัติการ Windows 11 / Windows 10
  2. โปรแกรม Microsoft Excel | Microsoft 365
